    public static isMod(client: Client, modRoles: ModRoleStorage, settings: SettingStorage, config: BotConfig, server: Server | string, user: User | string): boolean source

    Tests to see if a user is a moderator on a server using a specified client and config. If the server has not set any moderator roles, then they will be a moderator if any of their assigned roles contain the "Manage messages" permission. If the server has set moderator roles, then they will be a moderator if they have any of the moderator roles assigned. The bot owner and users with the "Administrate" permission are always moderators.
